# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## プロジェクト概要

跡見仁俊のパーソナルホームページ。チャットボット機能を備えたシングルページ構成。
現在は `index.html` のみのバニラHTML/CSS/JSプロジェクト。ビルドツール・パッケージマネージャーは使用していない。

## 言語・コミュニケーション

- **返答は必ず日本語で行う**
- コード内のコメントは日本語で書く

## デザイン原則

- **モダンで洗練されたデザイン**を維持する（ダークテーマ・グラデーション・グローエフェクト基調）
- **レスポンシブ対応必須** — すべての変更でスマートフォン表示を確認すること
  - ブレークポイントの目安: `max-width: 540px`（スマホ）、`max-width: 768px`（タブレット）
- CSS変数（`:root`）を使ってカラーテーマを一元管理する

## アーキテクチャ

現在はすべて `index.html` 1ファイルに集約されている。機能追加の際は以下の方針で分割を検討する:

```
my-hp/
├── index.html        # メインページ
├── style.css         # スタイル（分離する場合）
├── main.js           # スクリプト（分離する場合）
└── chatbot/          # チャットボット機能（今後追加予定）
```

## チャットボット機能（今後の実装方針）

- このプロジェクトの主要機能として **チャットボットをホームページに組み込む**
- UIはページ右下に固定表示するチャットウィジェット形式を想定
- バックエンドAPIとの連携が必要な場合は `fetch` を使用する
- チャット履歴はセッション中は保持する
